# ecommerce server-side


| Route         | HTTP   | Description               |
|---------------|--------|---------------------------|
| /carts/add    | POST   | add item to carts         |
| /carts/show   | GET    | show all carts            |
| /carts/remove | POST   | remove 1 item from carts  |
| /carts/clear  | DELETE | delete all items in carts |
